What is Milestone?

Milestone is an interactive learning and task tracker designed for students, developers, and lifelong learners. Instead of a flat to-do list, Milestone lets you structure your goals hierarchically, ensuring you can break down complex subjects into achievable, trackable steps.

Core Concepts

Workbooks

The highest level container. Think of this as a "Course", "Subject", or "Project" (e.g., Spring Boot Masterclass).

Categories

Logical groupings within a workbook to keep things organized (e.g., Beginner, Advanced).

Modules & Steps

A module is a specific topic containing actionable check-off steps (e.g., Setup Database module with steps to install and config).

Managing Workbooks

Creating a Workbook

From the main dashboard, click the New Workbook button. You will be prompted to provide:

  • Name: The title of your course or project.
  • Description: A brief overview of what this workbook covers.
  • Icon: A FontAwesome class (e.g., fa-brands fa-java) to represent the workbook visually.
  • Theme Color: Pick a custom color. The entire UI (progress bars, buttons, hover states) inside this workbook will adapt to this color!

Custom Terminology

One of Milestone's most powerful features is adapting to your workflow. In the Workbook creation modal, you can change what you call "Modules" and "Steps".

For a Coding Course
Module Term: Lab
Item Term: Checkpoint
For Reading a Book
Module Term: Chapter
Item Term: Page

Editing & Deleting

On the dashboard, hover over any workbook card. In the top right corner of the card, you will see a pencil icon (Edit) and a trash icon (Delete). Note: Deleting a workbook permanently removes all categories, modules, and progress inside it.

Categories & Modules

Once inside a Workbook, you build your curriculum using Modules, which are grouped by Categories.

Adding a Module

Click "Add New [Module Term]" in the sidebar. In the modal:

  1. Category/Section: Type a new category name (e.g., "Week 1") to create it, or select an existing one from the dropdown that appears as you type.
  2. Title & Icon: Name the specific module and give it an icon.
  3. Description: Explain the objective of this module.

Adding Steps & Using Markdown

At the bottom of the Module form, you add the actual checklist items. Each item requires a short title, but you can also add a highly detailed Description.

PRO TIP

Markdown is fully supported in step descriptions!

You can format text with `code blocks`, **bold text**, lists, and links. This is perfect for pasting command-line instructions, code snippets, or reference links right into your checklist.

Reordering: Use the and arrow buttons next to a step in the form to move it up or down in the list.

Managing Categories

On the Workbook's main overview screen (the Category View), you will see cards for each Category. Hovering over the category title reveals options to:

  • Rename the entire category.
  • Delete the category and all modules within it.

Progress Tracking

Milestone tracks your completion at three different levels automatically:

Module Level

In the sidebar, next to each module, you'll see a percentage (e.g., 50%). When all steps in a module are checked, it turns into a green checkmark.

Category Level

On the Workbook home screen, each Category card features a progress bar showing how many steps across all its modules have been completed.

Workbook Level

The main progress bar fixed at the top of the Workbook view tracks your overall completion of the entire course.

Data & Automation

Importing JSON to a Workbook

Instead of typing out massive courses manually, you can import data using a JSON array. Inside a Workbook, click the Import button in the sidebar footer.

example.json
[
  {
    "category": "Week 1",
    "labs": [
      {
        "title": "Introduction",
        "icon": "fa-solid fa-star",
        "description": "Welcome module",
        "checkpoints": [
          { 
            "text": "Read syllabus", 
            "description": "Find it on the portal." 
          }
        ]
      }
    ]
  }
]

Note: The JSON keys for the arrays (e.g., "labs", "checkpoints") dynamically adapt to whatever custom terminology you set for that workbook! Milestone is smart enough to find the arrays.

Global Backup & Restore

Because Milestone runs entirely in your browser (Local Storage), it's highly recommended to export your data periodically.

  • Click the Settings Gear icon in the top right header.
  • Click Export All Data to download a `.json` backup of all workbooks, modules, and completion statuses.
  • Use Import Data to upload that file to a different browser or device to restore your state.
  • Factory Reset completely wipes your local storage. Use with extreme caution!

Supercharge with AI

Don't want to type out a massive curriculum manually? You can use AI (like ChatGPT, Gemini, or Claude) to generate a complete course roadmap and import it into Milestone in seconds!

The AI Workflow

  1. Copy the Sample JSON: Go to your Milestone Workbook, click Import, and copy the "Sample JSON Format" displayed in the dropdown.
  2. Prompt the AI: Go to your favorite AI and paste the sample. Write a prompt like:
    "I am learning [Topic]. Create a 20-step learning roadmap for me. Output the result strictly using this JSON structure: [Paste JSON]."
  3. Copy & Paste: Copy the JSON code the AI gives you, return to Milestone, paste it into the Import text box, and click Import JSON.

✨ Just like magic, your workbook will instantly populate with categories, modules, and detailed checklists!

Creative Ways to Learn

Because you can change the terminology and use markdown in descriptions, Milestone is incredibly flexible. Here are a few ways you can customize it for different tasks:

Standard Learning Checklist

Break down a large subject (like learning a programming language or studying for an exam) into easily digestible chunks.

Module: Variables & Data Types
Step: Learn what a String is

Interview Q&A Flashcards

Use the expanding description area to hide answers. Test yourself by looking at the step title, then click to reveal the answer!

Step Title (Question): What is Closure in JS?
Description (Answer): [Hidden text explaining closure]

Interactive Quizzes

Make learning fun by turning a workbook into a quiz. Use the module as the question and the steps as options.

Module (Question): Which planet is closest to the Sun?
Step (Option A): Venus
Description: Incorrect! Venus is the second planet.

Daily Habit Tracker

Set up a workbook for "Morning Routine". Make the Modules the days of the week, and the steps your daily habits.

Module: Monday
Steps: Drink Water, Meditate, Read 10 Pages
